# ConsoleTweaks

ConsoleTweaks — это консольная утилита на Python, которая предоставляет улучшенный интерфейс для работы с командной строкой. Она поддерживает автодополнение команд, транслитерацию текста, выполнение системных команд и предоставляет интерактивное меню с дополнительными функциями.

## Основные возможности
- **Интерактивный интерфейс**: Удобный стартовый экран и поддержка автодополнения (при наличии библиотеки `prompt_toolkit`).
- **Команды программы**:
  - `help` — Показать список доступных команд.
  - `credits` — Показать информацию об авторах.
  - `changelog` — Показать историю изменений.
  - `version` — Показать текущую версию программы.
  - `cmdhelp` — Показать основные команды Windows CMD.
  - `translit <текст>` — Транслитерировать текст с латиницы на кириллицу.
  - `cls` — Очистить экран и показать баннер.
  - `stop` — Завершить работу программы.
- **Поддержка системных команд**: Выполнение любых системных команд (например, `dir`, `cd`, `copy` и т.д.).
- **Транслитерация**: Конвертация текста с латиницы на кириллицу (например, `privet` → `привет`).
- **Кроссплатформенность**: Работает на Windows, Linux и macOS.

## Требования
- Python 3.6+
- Библиотека `colorama` для цветного вывода в консоли.
- (Опционально) Библиотека `prompt_toolkit` для автодополнения и улучшенного ввода.

## Установка
1. Убедитесь, что у вас установлен Python 3.6 или выше.
2. Установите необходимые зависимости:
   ```bash
   pip install colorama
   pip install prompt_toolkit
   ```
3. Скачайте скрипт `console_tweaks.py` и поместите его в любую директорию.
4. Запустите скрипт:
   ```bash
   python console_tweaks.py
   ```

## Использование
1. Запустите программу, выполнив команду:
   ```bash
   python console_tweaks.py
   ```
2. При запуске отобразится стартовый баннер с подсказкой.
3. Введите команду:
   - Используйте встроенные команды, такие как `help`, `credits`, `translit <текст>` и т.д.
   - Выполняйте системные команды, например, `dir`, `cd <путь>`, `copy` и т.д.
   - Для выхода введите `stop` и подтвердите действие (`y` или `д`).
4. Для автодополнения (если установлен `prompt_toolkit`) используйте клавишу `Tab`.

## Примеры
- Показать список команд:
  ```
  help
  ```
- Транслитерировать текст:
  ```
  translit privet, kak dela?
  ```
  Результат: `привет, как дела?`
- Сменить директорию:
  ```
  cd C:\Users\User\Documents
  ```
- Выполнить системную команду:
  ```
  dir
  ```

## Зависимости

- **prompt_toolkit** (опционально): Для автодополнения и улучшенного ввода.
